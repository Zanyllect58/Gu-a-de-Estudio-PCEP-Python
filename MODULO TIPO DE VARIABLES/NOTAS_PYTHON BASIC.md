## Tipos de Variables en Python


```python
# Variables en Python

# int: Números enteros
# Se utilizan para representar números sin decimales. 
# Ejemplo: Edad, años, cantidad de elementos.
edad = 30  # La edad de una persona
anio = 2025  # Año actual o futuro

# float: Números decimales
# Se usan para representar números con decimales, como precios o tasas de interés.
# Ejemplo: Salarios, precios, tasas.
precio = 99.99  # Precio de un producto o servicio
interes = 0.05  # Tasa de interés anual del 5%

# str: Cadenas de texto (strings)
# Son secuencias de caracteres, como nombres, mensajes o descripciones.
# Ejemplo: Nombre de usuario, moneda, mensajes de bienvenida.
nombre = "Ana"  # Nombre de una persona
moneda = "USD"  # Moneda en la que se expresa el monto, en este caso dólares
mensaje = "Bienvenido a Finanzas con Python"  # Mensaje para la bienvenida

# bool: Valores booleanos (True o False)
# Se usan para representar condiciones o estados binarios.
# Ejemplo: Verificación de condiciones, si una inversión es segura o no.
es_mayor_de_edad = True  # Verificación si una persona es mayor de edad
inversion_segura = False  # Verificación si una inversión es segura o no

```
